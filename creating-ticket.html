Creating an Issue Ticket

Whenever you find an issue with your files, you can create an issue ticket which will then be taken care of by the technical support team.

Step One

Click on the icon on the top leftmost part beside WiSecure logo to enter the Create Ticket interface.

Step Two

Select the file or files with issue in column 1 (leftmost column).

A system log (center column) and list of File Issues (3rd column) will appear on your dashboard.

Step Three

Select an issue from the list available in the rightmost column.

A Case Reference number in the System Log column (center column) will be generated upon selecting an issue.

Step Four

Write a desciption of the issue on the textfield under the File Interaction Log column (rightmost column).

Step Five

Add a screenshot of the issue by clicking on the screenshot icon ().

The dashboard will return to the original layout. Click on the parts of the file that has found issues to put numbered labels.

Click the button to save the screenshot.

An (x) symbol on the screenshot icon () indicates an attached screenshot on the issue.

Step Six

Click the SEND button. The created issue with the screenshot will appear under the File Interaction Log column.
